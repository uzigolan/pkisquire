SCEP
Enterprise only		 SCEP GetCaps 
sscep getcaps -u http://localhost:8090/scep
curl http://localhost:8090/scep?operation=GetCACaps
 Query server capabilities (AES, DES3, POSTPKIOperation, SHA-256, etc.)
SCEP GetCA 
sscep getca \
  -u http://localhost:8090/scep \
  -c ca.crt
 Download CA certificate (prerequisite: RSA-based SubCA)
SCEP Enrollment 
sscep enroll \
  -u http://localhost:8090/scep \
  -k client.key \
  -r client.csr \
  -c ca.crt \
  -l client.crt \
  -E aes \
  -S sha256
 Enroll certificate via SCEP. Options: -E (encryption: aes|3des), -S (signature: sha256|sha512)
Challenge Password (API Token) 
curl -X POST \
  -H "Authorization: Bearer <API_TOKEN>" \
  http://localhost:80/api/challenge_passwords
 Generates a challenge password tied to the token's user. Response: value, expires_at, validity, user_id.
EST
Enterprise only		 EST Enrollment 
curl -k -X POST \
  --data-binary @etx.csr.der \
  https://pkisquire-ca.iot-rad.com/.well-known/est/simpleenroll \
  -H "Content-Type: application/pkcs10" \
  --output etx.crt.p7
estclient enroll \
  -server pkisquire-ca.iot-rad.com \
  -insecure \
  -csr etx.csr \
  -out etx.crt
 Using curl accepts DER CSR and returns signed cert in PKCS#7
