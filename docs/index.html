PKISquire CA

Flask-based Public Key Infrastructure (PKI) platform for certificate lifecycle management.

Supports SCEP, EST, and OCSP with optional post-quantum cryptography integrations.

  • Certificate issuance and revocation
  • CRL publication and OCSP validation
  • Web UI and REST API workflows
  • MQTT TLS enrollment support
