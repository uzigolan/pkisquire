# Skills

This directory contains reusable workflows for this repository.

## Available skills
- `test-suites`: Run the UI and API certificate test suites and the underlying PowerShell protocol scripts.
- `check-security`: Run Bandit, pip-audit, pip-licenses, and OpenSSL runtime capture; generate security/license reports and snapshot them into history.

## Usage
Open the relevant `SKILL.md` and follow the commands.
